# 2026-06-24 产品 UI 收口与个人中心本地计划

## 范围

- 新增并完善普通用户的“我的”页面，用于集中展示余额、免费标准章、最近任务、书架概览、快捷入口和最近余额记录。
- 从首页和书架移除账户余额、冻结金额、免费标准章等账户信息，避免普通页面重复展示财务摘要。
- 放大阅读器可用阅读区域：使用宽版外壳、缩窄左右侧栏、提高正文默认宽度并略收紧行高。
- 做一轮克制的产品 UI 打磨：统一按钮状态、卡片半径、页面背景层次、文字排版和普通任务页用语。
- 根据实际预览反馈，删除首页、上传页和创建译本页中过重的版权/隐私/公开体验提示，减少普通用户阅读干扰。
- 重做阅读器交互：默认只显示第一次翻译后的学习语言正文；每段提供“译”按钮，用户自行选择是否查看二次翻译；目录和 AI 阅读助手均可隐藏或显示；阅读主题支持白天、护眼和黑夜。
- 普通用户页面继续隐藏翻译点、token、模型、API、Provider、术语联网查证、术语本、成本、审计、数据保留、生产配置等内部概念。

## 非范围

- 不接入真实支付、真实 AI、真实后台队列、真实 Supabase 生产连接或真实对象存储。
- 不安装依赖、不联网、不读取或提交真实 `.env`。
- 不改变用户侧收费规则，仍保持 `0.5 元 / 标准章`，每个用户免费 `5` 个标准章。
- 不移动 Next.js 16 路由保护入口，仍保持在 `src/proxy.ts`。
